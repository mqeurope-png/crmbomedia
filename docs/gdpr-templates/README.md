# Plantillas de respuesta RGPD

Plantillas en español para responder formalmente a una solicitud RGPD.
**Antes de enviar**, completar los huecos `{{...}}` y validarlas con el
equipo legal / DPO.

Estructura de cada plantilla:

- Datos identificativos del responsable del tratamiento.
- Acción técnica realizada en el CRM (o motivo de rechazo).
- Plazo legal aplicado (1 mes prorrogable a 3).
- Vía de reclamación (AEPD u órgano competente).

Los emails se mandan **fuera del CRM** (no hay envío automático para
estas comunicaciones). Adjuntar, cuando aplique, el fichero generado
en `var/gdpr_exports/`.
